별도리 백엔드 API 문서에 오신 것을 환영합니다.

## Base URL

```
https://byeoldori-server-hbxnfn4woa-du.a.run.app
```

> **웹에서는** Vercel 프록시 `/api/[...path]`를 통해 호출합니다.

## 공통 응답 구조

```json
{
  "success": true,
  "message": "OK",
  "data": { ... }
}
```

## 인증

```
Authorization: Bearer {accessToken}
```

| 응답 코드 | 의미 |
|----------|------|
| `401` | 미인증 — 토큰 없거나 만료 |
| `403` | 권한 없음 |
